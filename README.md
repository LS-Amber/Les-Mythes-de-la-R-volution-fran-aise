# French Revolution Static Site

This folder contains a deploy-ready static version of the uploaded JSX game.

## Files
- `index.html`
- `styles.css`
- `app.js`

## Deploy to GitHub Pages
1. Create a new GitHub repository
2. Upload these three files to the repository root
3. In GitHub: Settings -> Pages
4. Source: `Deploy from a branch`
5. Branch: `main` and folder `/root`
6. Save
7. GitHub will generate a public URL for the site

## Notes
- No build step required
- Pure static HTML/CSS/JavaScript
- Fonts load from Google Fonts
